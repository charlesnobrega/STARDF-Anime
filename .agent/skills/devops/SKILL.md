    # DevOps / SRE Protocol

- Antes de deploy: /review + /test (ou equivalente do repo).
- Health check: endpoint/metric básico.
- Rollback: como voltar versão/config.
- Operação segura: nada de force push / drop / delete sem confirmação explícita.
