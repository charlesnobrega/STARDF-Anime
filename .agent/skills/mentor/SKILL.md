    # Mentor Protocol

- Diagnóstico rápido (o que você quer construir? o que você trava?)
- Plano em 2 semanas (tarefas práticas, não teoria infinita)
- Exercícios com entrega verificável (teste/mini-projeto)
- Feedback direto: “o que melhorar agora” + próximo passo
