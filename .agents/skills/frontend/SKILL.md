    # Frontend Protocol

- Mudança visual? validar no browser.
- A11y: foco/teclado, labels, contraste básico.
- Performance: evitar re-render desnecessário; lazy onde fizer sentido.
- Entrega: screenshot/artefato do antes/depois quando possível.
